# 🌸 UrCostome Edit Website

A handmade business website created for **UrCostome Edit**.

This website showcases crochet keychains, embroidery gifts, handmade products and customised gift items with an aesthetic design.

---

## Features

- Handmade business homepage
- Crochet & embroidery product gallery
- Product detail pages
- Customisation information
- Order form for customers
- Email notification system using FormSubmit
- Responsive design for mobile and desktop

---

##  Languages Used

- HTML
- CSS
- JavaScript

---

## Files Included

- index.html
- README.md
- Product images
- Logo image

---

## Order System

The website uses FormSubmit to receive customer orders through email.

Customers can:
- Select products
- Add custom details
- Submit orders

The business owner receives the order notification through email.

- Crochet Alphabet Keychains

- Cross Keychains
- Embroidery Initial Hoops

- Crochet Sunflower Gifts
- Bulk Handmade Orders

1. Save all files in one folder
